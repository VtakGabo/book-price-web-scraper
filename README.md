# book-price-web-scraper

Web scraping project that collects book prices and analyzes the dataset using pandas and data visualization.
This project scrapes book data from books.toscrape.com and analyzes book prices.

## Features

- Web scraping with BeautifulSoup
- Data cleaning and processing with pandas
- Data visualization with matplotlib
- Export scraped data to CSV

## Technologies

- Python
- requests
- BeautifulSoup
- pandas
- matplotlib
